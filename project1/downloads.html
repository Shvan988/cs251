Downloads

SYMBOLRT is open source software and is distributed under the GNU GPL license (version 3). The tool is only available for 32-bit Linux.

Sources are available at code.google.com/p/symbolrt and binaries can be downloaded here.

Dependences

  1. CVC3: an automatic theorem prover for Satisfiability Modulo Theories (SMT) problems used to decide on satisfiability of data guards;
  2. jgraphx: a tool used to show the generated TIOSTS models.
  3. XStream: a library to serialize the generated test cases to XML and back again.

If you have any trouble, visit the Support section to know how to proceed.
